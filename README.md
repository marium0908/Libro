# 📚 Libro Collective - Online Book Borrowing Platform

## 📌 Project Name
**Libro Collective - A Seamless and Modern Digital Library Experience**

---

## 🎯 Purpose
Libro is a modern online book borrowing platform designed to digitize the traditional library experience.

It allows users to explore, search, and borrow books from a curated digital collection with a focus on performance, accessibility, and user-friendly design.

---

## 🌐 Live URL
https://libro-collective.vercel.app/

---

## ✨ Key Features

- 📖 Curated Book Collection categorized by **Story, Tech, and Science**
- 🔍 Advanced Search & Category Filtering system
- 📚 Digital Borrowing system (Private Route)
- 👤 User Authentication (Email/Password + Google Login via BetterAuth)
- 📊 Personalized User Profile page
- 🖼️ Dynamic Homepage with SwiperJS slider
- 🔥 Featured Books section
- 🎯 Responsive design (Mobile, Tablet, Desktop)
- ✏️ Profile Update feature (Name & Photo URL)
- 🔔 Real-time toast notifications for actions

---

## ⚙️ Tech Stack

- **Next.js 15** - React framework
- **Tailwind CSS** - Styling
- **DaisyUI** - UI components
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **Swiper.js** - Carousel slider
- **React Hot Toast** - Notifications
- **BetterAuth** - Authentication system
- **MongoDB** - Database
