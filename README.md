# Libro - Online Book Borrowing Platform

## Project Name
**Libro Collective** - A Seamless and Modern Digital Library Experience.

## Purpose
Libro is designed to digitize the traditional library experience. It provides users with a curated digital sanctuary to explore, search, and borrow books from a decentralized repository. The platform focuses on accessibility, performance, and a distinctive aesthetic inspired by classical modern libraries.

## Key Features
- **Curated Repository**: A diverse collection of books categorized by Story, Tech, and Science.
- **Advanced Search & Filtering**: Find your next read easily using the title search bar and functional category sidebar.
- **Digital Borrowing System**: Borrow titles instantly with a single click (Private route for members).
- **Scholar Profiles**: Manage your borrowed volumes and personalize your scholar identity.
- **Dynamic Homepage**: Features handpicked library highlights (SwiperJS), real-time stats, and a scrolling marquee for updates.
- **Secure Authentication**: Integration with email/password and Google social login (BetterAuth logic).
- **Profile Insights**: Dedicated profile page showing "Active Readings" and user stats.
- **Identity Customization**: Update profile information (Name and Photo URL) with instant feedback.
- **Responsive Mastery**: Polished experience across mobile, tablet, and desktop devices using Tailwind Grid and Flexbox.
  
## NPM Packages Used
- **Next.js 15**: Core framework.
- **Tailwind CSS**: Utility-first styling.
- **DaisyUI**: Component library for polished UI elements.
- **Framer Motion**: Production-ready animations.
- **Lucide React**: Icon system.
- **Swiper.js**: Interactive featured books slider.
- **React Hot Toast**: Action confirmations and error handling.
- **BetterAuth**: Framework-agnostic authentication with MongoDB Adapter.
- **MongoDB**: Official driver for database connectivity.
